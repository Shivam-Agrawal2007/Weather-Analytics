I have created a Python based Weather Analytics System that fetches real-time weather information using the Open-Meteo API, stores weather history, performs statistical analysis, and generates visual reports and dashboards.

---

FEATURES:
- It can tell Weather of any city.
- It can tell forecast of 7 days.
- Saves weather history in a CSV file.
- Generate weather analytics like avd temp., max temp., min temp., humidity, etc.
- Search history based on cities.
- Compare weather of 2 cities.
- Generate charts.

---

Libraries and Other Sites that are used:
- Python
- Pandas
- NumPy
- Matplotlib
- Requests
- CSV
- Open-Meteo API

---

Output

The application generates:
- Weather history stored in CSV format
- Statistical weather analysis
- Temperature and humidity trend charts
- Weather distribution chart
- Dashboard containing multiple visualizations
- Text-based weather reports
- Application log file
